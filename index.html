Projeto: API de Adoção de Cães

Este projeto foi desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas. Trata-se de uma API RESTful criada para gerenciar um sistema de adoção de cães, simulando o funcionamento de uma plataforma real de cadastro e controle de animais disponíveis para adoção.

Utilizei Node.js no backend e MySQL como banco de dados, com estrutura completa de CRUD (Create, Read, Update, Delete) para os registros dos animais. A API permite operações como:

  • Cadastro de cães para adoção
  • Listagem com filtro e ordenação
  • Edição e exclusão de registros
  • Validação de dados

Todos os testes da API foram realizados com Insomnia, garantindo o funcionamento correto das rotas. Esse projeto me ajudou a consolidar conhecimentos em rotas, requisições HTTP, integração com banco de dados e organização de código em camadas.
